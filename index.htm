Going [rb-]gridfree

reboot's "gridfree" system moves away from the traditional 12 column grid based layout used in most responsive frameworks and opts for a more flexible and easy to configure method - relative scale.

relative scale is currently accomplished using a combination of CSS tricks and highly optimized script based methods to ensure compatibility across modern browsers and to minimize content popping post page load.
