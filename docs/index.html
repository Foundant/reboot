Going [rb-]gridfree

reboot's "gridfree" system moves away from the traditional 12 column grid based layout used in most responsive frameworks and opts for a more flexible and easy to configure method - relative scale.

relative scale is currently accomplished using a combination of CSS tricks and highly optimized script based methods to ensure compatibility across modern browsers and to minimize content popping post page load.

here is a set of div elements configured for relative scale using rb-gridfree data attributes. notice that fixed and relative width blocks can be mixed in the same container:

50
rel=3
rel=2
50
50
<div class="rb-container-dark rb-border-rounded rb-gridfree">
	<div data-gridfree-fix="50" class="rb-container">fix=150</div>
	<div data-gridfree-rel="3" class="rb-container">rel=3</div>
	<div data-gridfree-rel="2" class="rb-container">rel=2</div>
	<div data-gridfree-fix="50" class="rb-container">fix=150</div>
	<div data-gridfree-fix="50" class="rb-container">fix=150</div>
</div>

gridfree can be nested just like any other container. all the sizing still works relative to the siblings at the same level

rel=1 in rel=3
rel=2 in rel=1 in rel=3
rel=3 in rel=1 in rel=3
rel=1
<div class="rb-container-dark rb-border-rounded rb-gridfree">
	<div data-gridfree-rel="3" class="rb-gridfree">
		<div data-gridfree-rel="1" class="rb-container">
			rel=1 in rel=3
		</div>
		<div data-gridfree-rel="1" class="rb-gridfree">
			<div data-gridfree-rel="2" class="rb-container">
				rel=2 in rel=1 in rel=3
			</div>
			<div data-gridfree-rel="3" class="rb-container">
				rel=3 in rel=1 in rel=3
			</div>
		</div>
	</div>
	<div data-gridfree-rel="1" class="rb-container">rel=1</div>
</div>
