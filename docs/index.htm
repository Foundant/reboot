Going [rb-]gridfree

reboot's "gridfree" system moves away from the traditional 12 column grid based layout used in most responsive frameworks and opts for a more flexible and easy to configure method - relative scale.

relative scale is currently accomplished using a combination of CSS tricks and highly optimized script based methods to ensure compatibility across modern browsers and to minimize content popping post page load.

here is a set of div elements configured for relative scale using rb-gridfree data attributes. 1/3 of the parent is used for markup and 2/3 for a nested gridfree. notice that some grid blocks can be a relative size added to a fixed size: 

<div class="rb-gridfree">
    <div data-rb-gf-fix="50" data-rb-gf-rel="1">fix-50 + 1</div>
    <div data-rb-gf-rel="3">3</div>
    <div data-rb-gf-fix="100">fix-100</div>
    <div data-rb-gf-rel="2">2</div>
    <div data-rb-gf-fix="50" data-rb-gf-rel="1">fix-50 + 1</div>
    <div data-rb-gf-fix="150">fix-150</div>
</div>
